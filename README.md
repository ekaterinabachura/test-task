# Тестовое задание: Скрипт на Node.js для последовательной выгрузки 10,000 компаний из CRM Битрикс24 через REST API.

## Описание

Скрипт использует метод `crm.company.list` и механизм пагинации для обхода лимитов на выдачу данных (50 записей за запрос). Результат сохраняется в локальный JSON-файл.

## Запуск и проверка

1. **Клонируйте репозиторий:**
   ```bash
   npm install

2. **Конфигуркция вебхука:**
   Откройте файл index.js и замените заглушку на ваш рабочий вебхук Bitrix24 в переменной:
   ```bash
   const BITRIX_WEBHOOK_URL = 'ВАШ_ВЕБХУК_СЮДА/';

3. **Запуск скрипта:**
   Запустите скрипт через командную строку:
   ```bash
   node index.js

4. **Результат:**
После завершения работы скрипта (о чем будет уведомление в консоли), финальный список компаний будет сохранен в файле `result_companies.json`
